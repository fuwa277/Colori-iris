# Colori

一个专为数字绘画和设计开发的现代色彩管理与参考工具。

不仅仅是取色器，Colori 提供了一套完整的色彩工作流，包括高性能的屏幕监视（画中画）、基于物理按键模拟的跨应用吸色同步、以及符合人眼感知的灰度算法滤镜。

## 功能特性

### 🎨 高级调色与配色
- **多模式色轮**：支持传统的三角形/方形色轮，以及 RGB / HSV / CMYK / LAB 滑块。
- **同明度取色**：内置 Rec.601 亮度算法图表，可在锁定明度的情况下自由调整色相和饱和度，对绘画素描关系非常有帮助。
- **色彩方案**：实时生成互补色、近似色、对比色等配色方案，并支持一键随机寻找灵感。

### 🖥️ 屏幕监视器 (画中画)
- **高性能捕捉**：基于 Windows Graphics Capture (WGC) 技术，极低延迟，支持高帧率。
- **后台窗口监视**：即使目标窗口被遮挡，监视器依然能实时显示画面（需目标窗口非最小化）。
- **实时滤镜**：监视器窗口支持独立的灰度滤镜和镜像翻转，方便观察画面构成。
- **区域裁剪**：支持对监视源进行二次裁剪，只关注画布区域。

### 🔍 定点吸色与跨应用同步
- **定点监控**：锁定屏幕任意坐标，实时监测该点的像素颜色变化。
- **跨应用吸色 (Auto-Dropper)**：独家功能。在 Colori 中设置目标软件（如 Photoshop、CSP）的取色快捷键，即可实现“在 Colori 中按下快捷键 -> 自动激活目标软件 -> 自动吸取 Colori 当前色”的无缝流程。

### 🌗 全局灰度管理
- **双模式滤镜**：
  - **系统级**：一键触发 Windows 系统原生灰度滤镜 (Win+Ctrl+C)。
  - **应用级 (Mag API)**：使用 Windows Magnification API 实现的全屏灰度，支持自定义算法（Rec.601, BT.709, 平均值），比系统默认去色更符合视觉明度。

### 📌 参考图管理
- **浮窗参考**：支持从剪贴板或拖拽创建多个参考图窗口。
- **鼠标穿透**：参考图窗口支持“鼠标穿透”模式，覆盖在画布上方而不影响绘画操作。

## 技术栈

- **Frontend**: React, Tailwind CSS
- **Backend**: Rust (Tauri 2.0)
- **Windows APIs**:
  - Windows Graphics Capture (WGC) - 屏幕录制
  - Magnification API - 滤镜与色彩变换
  - GDI / User32 - 传统窗口管理与模拟输入

## 安装与运行

需要 Windows 10 (Build 1903+) 或 Windows 11 环境。

```bash
# 安装依赖
npm install

# 开发模式运行
npm run tauri dev

# 打包构建
npm run tauri build
```
